# Thu'um

**Thu'um** is a Skyrim-themed React component library inspired by [shadcn/ui](https://ui.shadcn.com). Instead of installing a package, you copy component source code directly into your project — giving you full ownership and customization.

## Why Thu'um?

- **Copy-paste architecture** — Components live in your codebase, not `node_modules`
- **Skyrim design tokens** — Dark, atmospheric color palette inspired by Tamriel
- **Built on standards** — Tailwind CSS v4, Radix UI primitives, CVA for variants
- **CLI-powered** — `npx thuum add button` fetches and installs components instantly

## Quick Start

```bash
npx thuum init
npx thuum add button
```

That's it. The component is now in your project, ready to customize.
