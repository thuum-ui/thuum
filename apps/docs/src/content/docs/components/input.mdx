## Usage

```tsx
  return <Input placeholder="Enter your name, Dragonborn..." />;
}
```

## Props

The `Input` component accepts all standard HTML input attributes.

```tsx
<Input type="email" placeholder="dovahkiin@skyrim.com" />
<Input disabled placeholder="Cannot edit" />
<Input type="password" placeholder="Secret passphrase" />
```
