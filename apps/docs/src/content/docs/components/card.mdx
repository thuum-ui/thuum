## Usage

```tsx
import {
  Card,
  CardCorners,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from "@/components/ui/card";

export function MyComponent() {
  return (
    <Card>
      <CardCorners />
      <CardHeader>
        <CardTitle>Quest Log</CardTitle>
        <CardDescription>Your active quests</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Defeat the World-Eater Alduin.</p>
      </CardContent>
      <CardFooter>
        <Button>Track Quest</Button>
      </CardFooter>
    </Card>
  );
}
```

## Sub-components

| Component         | Description                                          |
| ----------------- | ---------------------------------------------------- |
| `Card`            | Root container with top corner ornaments              |
| `CardCorners`     | Adds bottom corner ornaments (place as first child)   |
| `CardHeader`      | Top section with title/description                   |
| `CardTitle`       | Heading text (uppercase, tracked)                    |
| `CardDescription` | Secondary description text                           |
| `CardContent`     | Main content area                                    |
| `CardFooter`      | Bottom section with top border for actions            |
