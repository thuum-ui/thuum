## Usage

```tsx
import { Button } from "@/components/ui/button";

export function MyComponent() {
  return <Button variant="default">Fus Ro Dah</Button>;
}
```

## Variants

| Variant       | Description                                             |
| ------------- | ------------------------------------------------------- |
| `default`     | Text-only, dims when idle, brightens on hover (menu-style) |
| `primary`     | Bordered, visible at rest — for primary actions         |
| `destructive` | Red-tinted danger action                                |
| `outline`     | Bordered, text brightens on hover                       |
| `ghost`       | No border, subtle background on hover                   |
| `link`        | Underlined link style                                   |

## Sizes

| Size      | Description            |
| --------- | ---------------------- |
| `default` | Standard height (40px) |
| `sm`      | Compact (32px)         |
| `lg`      | Large (48px)           |
| `icon`    | Square icon button     |

## As Child

Use the `asChild` prop to render as a different element (e.g., a link):

```tsx
<Button asChild>
  <a href="/quests">View Quests</a>
</Button>
```
